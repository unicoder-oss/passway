# FAQ

## I forgot my password. What should I do?

Contact the administrator of your Passway installation or the person responsible for the organization. Do not create a new account unless it is necessary: access is tied to members and organizations.

## How do I share a secret with colleagues?

It is better to send a link to the secret rather than its value. This avoids the risk of leaking the secret through messenger or email, and the secret will be available only if the other user has access.

## Why can't I see a secret?

Either you are not a member of the required organization, or you (or a group you belong to) do not have permissions for the directory or secret.

## What if a secret may have been exposed?

Notify the person responsible for the organization, replace the secret value, and check the audit log. If an API key was involved, revoke it and create a new one.

## Where can I find the REST API documentation?

Open the [API](/api) page. It contains request and response examples for API applications.
