# Service capabilities

Passway lets you store confidential data (passwords, SSH keys, plain text) in one place with the ability to separate access to it between users.

## Main functionality

- Support for organizations for separate teams, projects, or areas of responsibility.
- Directories for separating secrets.
- Access restrictions for organization members and user groups.
- Secret generation from templates.
- Support for dynamic secrets with automatic rotation in external services.
- Automation through the REST API.
- Audit log with filtering support.
