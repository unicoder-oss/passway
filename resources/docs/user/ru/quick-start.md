# Быстрый старт

Начните работу с Passway и сохраните свой первый секрет.

## 1. Выберите организацию

На главной странице откройте нужную организацию. Обычно она соответствует команде, проекту или окружению.

Если в списке организаций пусто, попросите администратора создать организацию или прислать приглашение.

## 2. Создайте каталог

Каталоги помогают группировать секреты внутри организаций. Например, можно завести каталоги `Production`, `Staging`, `База данных` или `Внешние сервисы`.

На странице организации наведитесь на иконку "+" и нажмите на "Папку...":
![Начало создания каталога](/docs/images/ru/folder_create_start.png)

Откроется всплывающее меню для создания каталога:
![Основной процесс создания каталога](/docs/images/ru/folder_create_continue.png)

Здесь можно ввести любое название и указать права доступа к каталогу по умолчанию.

После создания откроется страница каталога, где можно добавлять секреты и при необходимости другие каталоги.

## 3. Добавьте секрет

Наведитесь на иконку "+" и нажмите на "Секрет...":
![Начало создания секрета](/docs/images/ru/secret_create_start.png)

В появившемся окне укажите имя и выберите тип секрета. Для примера создадим статический секрет:
![Продолжение создания секрета](/docs/images/ru/secret_create_continue.png)

Далее необходимо указать значение секрета: ввести вручную или использовать готовый шаблон. В примере введём значение вручную:
![Завершение создания секрета](/docs/images/ru/secret_create_final.png)

> [!NOTE]
> На этом шаге также можно указать права доступа к секрету по умолчанию. Можно выбрать, разрешается ли этот секрет читать и редактировать. По умолчанию используются права роли самого пользователя, но можно явно разрешить или запретить доступ любому участнику организации, вне зависимости от его роли.

После создания секрета откроется его страница, где можно раскрыть значение и выполнить действия над секретом:
![Страница секрета](/docs/images/ru/secret_reveal.png)
