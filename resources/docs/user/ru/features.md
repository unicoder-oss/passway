# Возможности сервиса

Passway позволяет хранить конфиденциальные данные (пароли, SSH-ключи, обычный текст) в едином месте с возможностью разграничить доступ к ним между пользователями

## Основной функционал

- Поддержка организаций для отдельных команд, проектов или зон ответственности
- Каталоги для разделения секретов
- Ограничение доступа к секретам для участников и групп пользователей
- Генерация секретов по шаблонам
- Поддержка динамических секретов с возможностью их автоматической ротации во внешних сервисах
- Автоматизация действий с помощью REST API
- Журнал действий с поддержкой фильтрации
