# Генерация, шаблоны и ротация

Вы можете хранить не только произвольные текстовые значения, но и генерировать новые секреты автоматически по шаблонам. Это снижает риск слабых или повторно используемых значений (например, паролей).

## Шаблоны

Шаблон задаёт правила генерации. На данный момент доступны шаблоны для генерации паролей и пар SSH-ключей (Ed25519 и RSA-4096)

## Когда использовать генерацию

- Нужен новый пароль для сервиса
- Нужно исключить ручной выбор слабого значения
- Команда хочет использовать единые требования к длине и символам

## Ротация

Ротация — это замена секрета новым значением. Она полезна после инцидента, смены участников команды или по регулярному графику.

> [!NOTE]
> Перед ротацией проверьте, какие системы используют секрет. После смены значения их может потребоваться обновить. Вместо ручной замены значений после ротации вы можете использовать REST API для получения актуального значения секрета приложениями и скриптами.

## История версий

Passway хранит сведения о версиях и действиях с секретом. Это помогает понять, когда значение было изменено, и каким способом.
